À propos du projet

Ce site a été réalisé dans le cadre du module TI402, pour sensibiliser à la préservation de la vie terrestre, l’un des 17 objectifs du développement durable de l’ONU.

Notre équipe

Nom Prénom 1

Recherche documentaire, rédaction des articles sur la biodiversité.

Nom Prénom 2

Développement web, design graphique, intégration responsive.

Nom Prénom 3

Interactivité, animations JavaScript, gestion du formulaire de contact.

Le projet en détails

  • Utilisation de HTML5, CSS3 (Flexbox, Grid, media queries) et JavaScript
  • Site responsive, adapté à tous les écrans
  • Navigation fluide et intuitive
  • Contenus originaux et images libres de droits (Unsplash, Pixabay...)
  • Code versionné sur GitHub
